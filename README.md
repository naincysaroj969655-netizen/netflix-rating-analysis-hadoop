Netflix Rating Analysis using Hadoop

Project Overview

This project analyzes the Netflix titles dataset using the Hadoop ecosystem in Cloudera.

The project uses:
    HDFS for distributed storage
    MapReduce for data processing
    Hive for SQL-based analysis
    Python for mapper and reducer scripts

The main objective is to count the number of Netflix titles in each rating category and identify the most common ratings.

Dataset
Dataset used: netflix_titles.csv
Important fields:
show_id
type
title
country
release_year
rating
duration
listed_in
